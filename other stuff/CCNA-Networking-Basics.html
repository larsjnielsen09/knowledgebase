OSI Model

  • Physical Layer Deals with physical transmission of data
  • Data Link Layer Handles error-free transfer between nodes
  • Network Layer Manages addressing and routing
  • Transport Layer Ensures reliable data delivery

IP Addressing

  • IPv4 32-bit addressing system
  • Subnetting Division of IP networks into smaller networks
  • CIDR Classless Inter-Domain Routing

Network Protocols

  • TCP/IP Transmission Control Protocol/Internet Protocol
  • UDP User Datagram Protocol
  • ICMP Internet Control Message Protocol
